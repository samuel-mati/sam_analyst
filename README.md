⚙️ Getting Started
To run the project locally, follow these steps:

Clone this repository:
```bash
git clone https://github.com/samuel-mati/sam_analyst.git
```

Navigate into the project directory:
```bash
cd sam_analyst
```

Install dependencies:
```bash
npm install
```

Run the project:
```bash
npm run dev
```
